## Maps

``Maps`` _are_ ``Collections`` containing only entries with unique keys.
- add value for key
- set value for key
- replace value for key
- remove value for key
- add all entries
- set all entries
- replace all entries
- remove all entries for key
- keep all entries matching keys
- merge entries
- remove given entry
- remove all entries
- contains key
- contains all keys
- get value for key
- get entry for key
- ``Set`` view of keys
- ``ReducibleLists`` view of values
